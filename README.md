# RGB Defense

빛의 삼원색(RGB)을 소재로 한 탑뷰 타워디펜스 게임

## 게임 소개

적들은 RGB 체력을 가지고 있습니다. 같은 색의 타워로 공격하면 100% 피해를, 다른 색의 타워는 33%만 입힙니다. 전략적인 타워 배치와 색상 조합이 승리의 핵심입니다.

### 조작

| 키 | 동작 |
|---|---|
| `R` | Warrior 선택 |
| `G` | Archer 선택 |
| `B` | Mage 선택 |
| `Space` | 웨이브 시작 |
| `ESC` | 일시정지 / 취소 |
| 마우스 우클릭 | 타워 선택 취소 |

## 타워

각 타워는 1차 → 2차 전직을 통해 강력한 특수 능력을 얻습니다.

### 🔴 Warrior — 근접 공격 (20코인)

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 검성 | 검기 발사 | 검존 | 검기 폭 증가 + 관통 |
| 바바리안 | 연속 공격 후 휴식 | 광전사 | 연속 공격 강화, 체력 낮을수록 공속 ↑ |
| 격투가 | 3타마다 넉백 | 투신 | 2타마다 넉백, 기절 |

### 🟢 Archer — 원거리 공격 (30코인)

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 저격수 | 관통 화살 | 데드아이 | 전 대상 관통, 사거리 ↑ |
| 사냥꾼 | 화살 3발 확산 | 폭풍 | 화살 5발, 공속 ↑ |
| 독사 | 슬로우 화살 | 바실리스크 | 강력한 슬로우 + 도트 |

### 🔵 Mage — 원거리 스플래시 (40코인)

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 번개술사 | 체인 라이트닝 | 벽력 | 연쇄 9회, 데미지 증가 |
| 네크로맨서 | 해골 소환 | 죽음군주 | 해골 2마리, 사망 시 폭발 |
| 빙결학자 | 적 빙결 | 절대영도 | 빙결 시간 증가, 범위 빙결 |
