# RGB Defense 기획

빛의 삼원색(RGB)을 소재로 한 탑뷰 타워디펜스 게임

## 게임 디자인

### 적

- vector3인 rgb(현재 체력), max_rgb(최대 체력) 보유
- max_rgb로 가능한 값
  - 단일 색(1.0, 0.0, 0.0) red, green, blue
  - 혼합 색(1.0, 1.0, 0.0) yellow, aqua, magenta
- 각 색은 rgb / max_rgb 로 표시하되, rgb가 0.0 초과일 경우 표시되는 색은 최소 0.4 보정
- 모든 색 값이 0.0이 되면 제거됨
- 체력 바 표시
  - 적 위에 직사각형 체력 바 1개를 표시
  - 감소한 체력은 오른쪽 일부에 검은 색으로 표시
  - 현재 체력은 왼쪽 일부에 rgb의 값을 정규화한 색으로 표시 (예: (0.0, 0.1, 0.2) -> (0.0, 1.0, 1.0))
- 적을 누르면 세부 정보를 확인할 수 있음

### 타워

- 빨강 타워, 초록 타워, 파랑 타워 존재
  - 빨강 타워: 전사, 근접 공격
    - 검성: 검기를 날려 원거리 공격
    - 바바리안: 짧은 시간 빠른 속도로 연속 공격, 공격 후 지쳐서 잠시 쉼
    - 격투가: 몇 번 공격할 때마다 넉백을 일으킴
  - 초록 타워: 궁수, 원거리 공격
    - 신궁: 거대한 관통 활
    - 폭풍궁: 한 번에 화살 3개 발사
    - 독사: 독화살을 날려 적을 일정 시간 느려지게 만듦
  - 파랑 타워: 마법사, 원거리 스플래시 공격
    - 벽력: 체인 라이트닝
    - 네크로맨서: 사정거리에 적이 있다면 짧은 시간 유지되는 해골을 소환함, 해골은 트랙에서 적을 따라가며 거리에 제한이 없음
    - 서리술사: 적을 얼림
- 타워는 그리드 방식으로 배치 가능
- 타워가 적을 공격하면 적의 rgb 값 중 타워 색의 값이 감소함
- 각 타워는 적 중 자기 자신에 해당하는 rgb 값이 0.0 초과인 적만 공격 가능함
- 타워를 클릭하면 세부 정보를 확인할 수 있음

### 트랙

- 트랙은 직선으로 구성되어 있음
- Path2D로 구현

### UI

- 좌측 상단에 생명(lives)과 돈(coins), 웨이브 표시
- 좌측 하단에 웨이브 프리뷰(예: 빨강x3, 초록x1)와 Start Wave 버튼 표시
- 우측에 타워 목록 표시
