# RGB Defense 기획

빛의 삼원색(RGB)을 소재로 한 탑뷰 타워디펜스 게임

## 게임 디자인

### 적

- [x] vector3 rgb(현재 체력), max_rgb(최대 체력) 보유
- [x] max_rgb로 가능한 값: 단일 색 (red, green, blue)
- [ ] 혼합 색 (yellow, aqua, magenta) — 미구현
- [x] 각 색은 rgb / max_rgb 로 표시하되, rgb가 0.0 초과일 경우 표시되는 색은 최소 0.4 보정
- [x] 모든 색 값이 0.0이 되면 제거됨
- [x] 체력 바 표시
- [ ] 적을 누르면 세부 정보를 확인할 수 있음 — 미구현

### 타워

- [x] 빨강 타워 (Warrior), 초록 타워 (Archer), 파랑 타워 (Mage)
- [x] 타워는 그리드 방식으로 배치 가능
- [x] 타워가 적을 공격하면 적의 rgb 값 중 타워 색의 값이 감소, 다른 색에는 1/3의 피해만을 적용
- [x] 타워는 동일 색상 적 우선 타겟팅, 없으면 가장 앞선 적 타겟팅
- [x] 타워 클릭 시 판매(구매가 90% 환불) 및 전직 메뉴 표시
- [x] 1차 전직은 타워당 3개 중 1개 선택
- [x] 2차 전직은 1차 전직 특성 극대화

#### 빨강 타워: 전사 (Warrior) — 근접 공격

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 검성 (Sword Saint) | 근거리 공격시 검기를 날림 | 검존 (Sword Sovereign) | 검기 폭 증가 + 관통, 사거리 증가 |
| 바바리안 (Barbarian) | 짧은 시간 빠른 속도로 연속 공격, 공격 후 휴식 | 광전사 (Berserker) | 연속 공격 시간 증가, 휴식 감소, 적 체력 낮을수록 공속 증가 |
| 격투가 (Brawler) | 3회 공격마다 넉백 | 투신 (Battle God) | 2회 공격마다 넉백, 넉백 거리 증가, 넉백된 적 기절 |

#### 초록 타워: 궁수 (Archer) — 원거리 공격

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 저격수 (Sniper) | 큰 화살이 적을 관통 | 데드아이 (Deadeye) | 전 대상 관통, 첫 타겟 치명타, 사거리 증가 |
| 사냥꾼 (Hunter) | 한 번에 화살 3개 발사 | 폭풍 (Tempest) | 화살 5발 확산, 공속 증가 |
| 독사 (Viper) | 독화살로 적에게 슬로우 | 바실리스크 (Basilisk) | 슬로우 + 도트 데미지, 독 전염 |

#### 파랑 타워: 마법사 (Mage) — 원거리 스플래시 공격

| 1차 전직 | 설명 | 2차 전직 | 설명 |
|---|---|---|---|
| 번개술사 (Electromancer) | 체인 라이트닝 | 벽력 (Thunderbolt) | 연쇄 9회, 체인 분기, 연쇄당 데미지 증가 |
| 네크로맨서 (Necromancer) | 해골 소환, 해골은 근접 공격 | 죽음의 군주 (Death Lord) | 해골 2마리 소환, 해골 강화, 사망 시 폭발 |
| 빙결학자 (Cryomancer) | 적을 얼림 | 절대영도 (Absolute Zero) | 빙결 시간 증가, 빙결된 적 추가 피해, 범위 빙결 |

- [x] 키보드 단축키: Space(웨이브 시작), R/Warrior, G/Archer, B/Mage
